# Определение

Аббревиатура CDMA расшифровывается как Code Division Multiple Access — «множественный доступ с кодовым разделением». Это технология радиосвязи, при которой все абоненты используют одну и ту же полосу частот одновременно, но каждый сигнал кодируется уникальной псевдослучайной последовательностью [@wiki_cdma_2026].
В отличие от традиционных методов разделения каналов по частоте (FDMA) или времени (TDMA), CDMA позволяет каждому пользователю занимать весь спектр и всё время соединения, что обеспечивает высокую спектральную эффективность и помехоустойчивость (рис. [-@fig:001]).

![Схема-сравнение FDMA, TDMA и CDMA](./image/FDMA-TDMA-CDMA-Frequency-Time-division.png){#fig:001 width=70%}

## Историческая справка

Идеи кодового разделения каналов появились задолго до эры сотовой связи. В СССР первая научная работа по этой теме была опубликована ещё в 1935 году профессором Дмитрием Агеевым. Однако практическое применение технология получила лишь в конце 1980-х годов благодаря компании Qualcomm [@ccf_cdma_history_2007].

* 1989 г. — публичная демонстрация цифровой сотовой системы на базе CDMA 
* 1993 г. — принятие стандарта IS-95 (cdmaOne) как первого коммерческого стандарта 2G на основе CDMA
* 1995 г. — запуск первой коммерческой сети в Гонконге

С тех пор технология эволюционировала: от базового cdmaOne к семейству стандартов CDMA2000, поддерживающих высокоскоростную передачу данных третьего поколения (3G).

# Принцип работы технологии

Каждому абоненту присваивается уникальный ортогональный код (шумоподобная последовательность). При передаче данные комбинируются с этим кодом с помощью операции побитового исключающего ИЛИ (XOR), что «расширяет» спектр сигнала [@skynet_cdma_lecture_2009; @geeksforgeeks_is95_2025].

Как это работает на практике:

1. Передача: сигнал абонента «размазывается» по широкой полосе частот с использованием индивидуального кода.
2. Приём: базовая станция, зная код конкретного пользователя, выделяет его сигнал из общей «какофонии» путём корреляционной обработки 
3. Помехоустойчивость: посторонние сигналы (включая сигналы других абонентов) воспринимаются как шум и подавляются.

Важные технические особенности:

- Фактор расширения спектра (отношение длительности бита данных к длительности чипа кода) определяет потенциальную ёмкость канала.
- Требуется точная синхронизация всех передатчиков по времени — это одно из условий корректной работы системы.
- Мощность передатчиков адаптивно регулируется, чтобы «близкие» абоненты не заглушали «дальних» (проблема «ближний-дальний»).

## Преимущества технологии

В таблицах [-@tbl:tabl1] и [-@tbl:tabl2] представлены достоинства и недостатки технологии [@euromobile_cdma_principle_2018]. 

: Преимущества {#tbl:tabl1}

| Преимущество | Описание |
|------------------------|----------|
| **Высокая ёмкость** | В одном частотном канале может работать в 4–5 раз больше абонентов, чем в GSM |
| **Помехоустойчивость** | Шумоподобная структура сигнала делает систему устойчивой к узкополосным помехам и перехвату |
| **Качество речи** | Использование адаптивных кодеков (13 кбит/с) и технологии мягкого переключения (soft handoff) |
| **Энергоэффективность** | Низкая пиковая мощность излучения (до 250 мВт) продлевает время работы батареи |
| **Гибкость ресурсов** | Нет жёсткого лимита каналов: при росте нагрузки качество плавно снижается |

## Недостатки

: Недостатки {#tbl:tabl2}

| Недостаток | Описание |
|------------------------|----------|
| **Сложность реализации** | Требуются мощные процессоры для обработки сигналов в реальном времени |
| **Зависимость от синхронизации** | Сбои в системе синхронизации (GPS) могут нарушить работу всей сети |
| **Ограниченная совместимость** | Меньшее глобальное покрытие по сравнению с GSM |

# Развитие технологии

Технология CDMA не стояла на месте. Её развитие шло по пути увеличения скорости передачи данных [-@tbl:tabl3] [@ixbt_cdma_overview_2000].

: Развитие {#tbl:tabl3}

| Технология | Принцип разделения | Эффективность | Применение |
|------------|-------------------|---------------|------------|
| **FDMA** | По частоте | Низкая | Аналоговая связь (1G) |
| **TDMA** | По времени | Средняя | GSM, D-AMPS (2G) |
| **CDMA** | По коду | Высокая | IS-95, CDMA2000 (2G-3G) |
| **OFDMA** | По ортогональным поднесущим | Очень высокая | LTE, 5G (4G-5G) |

Важно: все эти стандарты сохраняют совместимость на уровне базовых принципов кодирования, что позволяло операторам модернизировать сети поэтапно.

# Заключение

Подводя итог, отмечу: технология CDMA стала важным этапом в развитии мобильной связи. Она доказала эффективность кодового разделения каналов, обеспечила качественный скачок в ёмкости и защищённости сетей. Хотя сегодня доминируют стандарты на основе OFDMA (LTE, 5G), принципы CDMA продолжают использоваться в гибридных решениях и остаются актуальными для специализированных систем связи.
